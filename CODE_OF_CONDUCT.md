# Code of Conduct

LewLM aims to keep this project respectful, constructive, and welcoming.

## Expected behavior

- Be respectful and professional.
- Assume good intent and discuss disagreements in good faith.
- Give actionable technical feedback.
- Respect project boundaries, contributor time, and review decisions.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks
- Trolling, intimidation, or sustained disruption
- Publishing private information without permission
- Sexualized language or unwelcome sexual attention

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions
that violate this code of conduct and may limit future participation when needed
to protect the project.
